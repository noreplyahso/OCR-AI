Before using it on a production machine, verify these prerequisites:

1. Basler pylon runtime/driver must be installed if the machine will connect to a Basler camera.

3. MySQL must be available if the app still uses the local drb_text database configuration.
4. Product Excel data and runtime models must remain inside the installed application folder.

- %LOCALAPPDATA%\DRB-OCR-AI\logs\app.log
